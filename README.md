# akt_card-detection

## Usage

### Create a virtual environment

```Powershell
python -m venv <venv_name>
.\<venv_name>\Scripts\activate
```

### Installation

```Powershell
pip install -r requirements.txt
python -m ipykernel install --user --name=<venv_name>
```

### Open jupyterlab

```Powershell
python -m jupyterlab
```
